# Qwasar Silicon Valley
## my_printf project
 
This project is to rewrite the printf function in C. 
 

### Compiling

Run the following commands:

* To compile
	- `make`
* To remove objects:
	- `make clean`
* To remove objects and binary file (program):
	- `make fclean`
* To re-compile:
	- `make re`
